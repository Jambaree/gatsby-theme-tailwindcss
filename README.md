# Repo for gatsby-theme-tailwindcss

## How to use this repo

1.  Clone the repo
2.  Install dependencies

    ```sh
    yarn
    ```

    or

    ```sh
    npm install
    ```

3.  Start the demo site

    ```sh
    yarn start
    ```

    or

    ```sh
    npm start
    ```
